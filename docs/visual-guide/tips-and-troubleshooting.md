## Tips & Troubleshooting

### Finding events in Discord

1. Click the server name at top-left → **Events**
2. Or look for the calendar icon in the channel list

[SCREENSHOT]

### Checking event details

1. Click on an event in the Events list
2. View description, banner, start time (in your local timezone), and location

[SCREENSHOT]

### Checking bot permissions

1. Server Settings → Roles → find the bot's role → ensure **Manage Events** is enabled
2. For Voice/Stage channels: right-click channel → Edit Channel → Permissions → check the bot role

[SCREENSHOT]

### If commands don't appear

1. Type `/` — do you see the command menu?
2. If missing: wait 5–10 minutes, refresh Discord (Ctrl+R), and check the bot is online

### If buttons don't respond

Click again — Discord UI can be laggy. Refresh if needed.

### If events don't appear

1. Wait a few minutes — events are created gradually
2. Run `/schedule list` to confirm the schedule is active
3. Check Discord's Events section (not just the channel list)
4. Verify the bot has permissions
